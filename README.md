# checknetwork

Simple tool to quickly check various endpoints on your network in one simple command

`checknetwork`
```
LAN: 192.168.1.32 (hostname)
  -> Loopback: 127.0.0.1        |  1ms  |
  -> Gateway : 192.168.1.1      |  7ms  | 
  -> ISP     : 123.123.123.123  | 12ms  | 
  -> DNS     : 8.8.8.8          | 37ms  |
  -> AWS     : 1.2.3.4          | 39ms  |
```
